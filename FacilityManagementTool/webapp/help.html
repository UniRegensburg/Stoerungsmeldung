Anmelden

Hier können Sie sich mit ihrem gültigen NDS-Acount anmelden. Dies ist notwendig um eine Störung melden zu können.

Anmeldedaten

Hier können Sie ihre persönlichen Daten wie Name, Telefonnummer und Emailadresse eingeben und korrigieren. Diese Daten sind notwendig, um bei eventuell auftretenden Problemen bei der Behebung von Störungen mit ihnen Kontakt aufnehmen zu können.

Einschätzung

Hier können Sie entscheiden, ob Sie eine dringende Störung (Rohrbruch, Heizungsdefekt) telefonisch oder eine normale Störung über die App melden möchten.

Dringende Störung

Hier können Sie der technischen Zentrale eine dringende Störung (Rohrbruch, Heizungsdefekt) telefonisch melden.

Störungsmeldung

Hier können Sie eine Störung per Störungsformular melden. Geben Sie hierfür die benötigten Informationen zum Raum, zur betroffenen Fachgruppe und zur Störung an sich ein und melden Sie die Störung anschließend mithilfe des "Störung melden" Buttons. Falls Sie der Meldung multimediale Inhalte (Bild-, Pdf-, Excel-Dateien) beifügen möchten, aktivieren Sie den entsprechenden Schalter (Sie werden anschließend auf eine entsprechende Seite weitergeleitet) .

Störungsanhang

Hier können Sie der Meldung multimediale Inhalte (Bild-, Pdf-, Excel-Dateien) beifügen. Nehmen Sie per "Neues Bild machen"-Button neue Bilder auf, welche Sie anschließend per "Anhang senden"-Button an die technische Zentrale schicken können. Hierbei wird die Standard-Email-Anwendung ihres Gerätes mit vordefiniertem Empfänger und Betreff geöffnet. Nun können Sie ihre Anhänge auswählen und versenden.
