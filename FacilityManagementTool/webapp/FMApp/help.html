Login

To claim a disturbance, you need to log in with a valid NDS-Account, which consists of three letters and five numbers.

Login information

Additionally, you need to give your phone number to be available for a call from the technical support for any further information.

Estimation

If you want to claim an urgent disturbance you must choose the red button to get redirected to your phone app. If that is not the case, choose the green button where you are prompted to fill the form for a normal disturbance claim.

Urgent disturbance

An urgent disturbance is a disturbance, which has to be fixed immediately because it dangers people or buildings.

Fault report

For the fault report you need to fill in the information about the disturbance, the afforded room and the expert group which is expected to fix the disturbance. Optionally, you can append an image-file (Jpeg, Jpg and Png are supported) to add further information about the disturbance. Afterwards, you just need to press the "Claim disturbance"-Button.

Disturbance attachement

Now you can attach image files to your disturbance report. Choose files via the according button and upload them by clicking the "Submit attachement"-Button. The max file size is 5 MB.

Anmelden

Hier können Sie sich mit ihrem gültigen NDS-Acount anmelden. Dies ist notwendig um eine Störung melden zu können.

Anmeldedaten

Hier können Sie ihre persönlichen Daten wie Name und Telefonnummer eingeben und korrigieren. Diese Daten sind notwendig, um bei eventuell auftretenden Problemen bei der Behebung von Störungen mit ihnen Kontakt aufnehmen zu können.

Einschätzung

Hier können Sie entscheiden, ob Sie eine dringende Störung (Rohrbruch, Heizungsdefekt) telefonisch oder eine normale Störung über die App melden möchten.

Dringende Störung

Hier können Sie der technischen Zentrale eine dringende Störung (Rohrbruch, Heizungsdefekt) telefonisch melden.

Störungsmeldung

Hier können Sie eine Störung per Störungsformular melden. Geben Sie hierfür die benötigten Informationen zum Raum, zur betroffenen Fachgruppe und zur Störung an sich ein und melden Sie die Störung anschließend mithilfe des "Störung melden" Buttons. Falls Sie der Meldung Bilddateien (Jpeg, Jpg und Png werden unterstützt) beifügen möchten, aktivieren Sie den entsprechenden Button (Sie werden anschließend auf eine entsprechende Seite weitergeleitet). Betätigen Sie zum Abschluss den "Störung melden"-Button.

Störungsanhang

Hier können Sie der Meldung Bilddateien (Jpeg, Jpg und Png werden unterstützt) beifügen . Wählen Sie die gewünschten Anhänge mithilfe des "Anhang auswählen"-Button aus und laden Sie diese per Druck auf den "Anhang abschicken"-Button aus. Die maximale Dateigröße ist begrenzt auf 5MB.
