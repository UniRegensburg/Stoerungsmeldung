Login

To claim a malfunction, you need to log in with a valid NDS-Account, which consists of three letters and five numbers.

Login information

Additionally, you need to give your phone number to be available for a call from the technical support for any further information.

Estimation

If you want to claim an urgent fault you must choose the red button to get redirected to your phone app. If that is not the case, choose the green button where you are prompted to fill the form for a normal malfunction claim.

Urgent malfunction

An urgent malfunction is a malfunction, which has to be fixed immediately because it dangers people or buildings.

malfunction report

For the malfunction report you need to fill in the information about the malfunction, the afforded room and the expert group which is expected to fix the malfunction. Optionally, you can append image-files (Jpeg, Jpg and Png are supported) to add further information about the malfunction. Click the "Add-picture" button to add image files to the upload list below. These files will be uploaded to the server. To delete files from the list click the icon next to the according file name. The max file size is 5 MB.

Anmelden

Hier können Sie sich mit ihrem gültigen NDS-Acount anmelden. Dies ist notwendig um eine Störung melden zu können.

Anmeldedaten

Hier können Sie ihre persönlichen Daten wie Name und Telefonnummer eingeben und korrigieren. Diese Daten sind notwendig, um bei eventuell auftretenden Problemen bei der Behebung von Störungen mit ihnen Kontakt aufnehmen zu können.

Einschätzung

Hier können Sie entscheiden, ob Sie eine dringende Störung (Rohrbruch, Heizungsdefekt) telefonisch oder eine normale Störung über die App melden möchten.

Dringende Störung

Hier können Sie der technischen Zentrale eine dringende Störung (Rohrbruch, Heizungsdefekt) telefonisch melden.

Störungsmeldung

Hier können Sie eine Störung per Störungsformular melden. Geben Sie hierfür die benötigten Informationen zum Raum, zur betroffenen Fachgruppe und zur Störung an sich ein und melden Sie die Störung anschließend mithilfe des "Störung melden" Buttons. Optional können Sie der Meldung Bilddateien (Jpeg, Jpg und Png werden unterstützt) beifügen möchten, um die Störung näher zu beschreiben. Wählen Sie die gewünschten Anhänge mithilfe des "Bild hinzufügen"-Button aus. Diese werden anschließend in die Liste der Dateien, welche mit ihrer Störungsmeldung and den Server geschickt werden augenommen. Um Bilder aus dieser Liste zu löschen, klicken Sie auf das neben dem jeweiligen Bildnamen liegende Icon. Die Größe einzelner Bilder ist auf 5 MB begrenzt.
